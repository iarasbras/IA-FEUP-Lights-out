Lights Out

Start Menu

Choose your board size and jump right into a puzzle

1

Pick a board

Start with 3×3 if you want something quick, then move up to 4×4 or 5×5 for a harder puzzle.

2

Click lights

Press a cell to toggle it plus the four adjacent cells in a cross pattern.

The board is solved when every square is dark, so you can use the pattern to work out the moves step by step.

3

Finish the grid

Keep going until all lights are off. If you get stuck, use Ask Hint or try an AI solver.

Tip: Order does not matter, so you can experiment freely.
Tip: Clicking the same cell twice cancels itself out.
Tip: If you get stuck, try solving row by row from the top, then use the AI solver or a hint.
